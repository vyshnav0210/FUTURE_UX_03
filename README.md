# 🚀 ClientFlow CRM – Customer Relationship Management Dashboard

## 📌 Project Overview

ClientFlow CRM is a modern Customer Relationship Management (CRM) dashboard designed in Figma to help businesses efficiently manage leads, clients, revenue, tasks, and business performance through a clean and intuitive user interface.

## 📸 Screenshot

### Dashboard Overview

![Dashboard](Screenshots/Dashboard.png)

---



The dashboard allows users to:

* Track Total Leads
* Manage Active Clients
* Monitor Revenue Growth
* Track Business Tasks
* View Analytics Reports
* Monitor Recent Deals
* Access Business Insights
* Manage CRM Activities

## 🎯 Features

* Dashboard Overview
