CLIENTFLOW CRM DASHBOARD UI/UX DESIGN

## Project Overview

ClientFlow CRM is a modern Customer Relationship Management (CRM)
dashboard designed in Figma to help businesses manage leads,
clients, revenue, tasks, and business performance efficiently.

The dashboard provides a centralized platform for monitoring
business activities and customer relationships through a clean,
professional, and data-driven user interface.

## Features

1. Dashboard Overview
2. KPI Performance Cards
3. Revenue Analytics
4. Revenue Donut Chart
5. Recent Deals Table
6. Search Functionality
7. Sidebar Navigation
8. User Profile Section

## Dashboard Metrics Included

* Total Leads
* Active Clients
* Revenue Tracking
* Task Management
* Deal Monitoring
* Business Analytics
